# 南京大学选课小助手 (NJU选课Helper)

![](icons/icon128.png)

## 项目简介

南京大学选课小助手是一款专为南京大学学生设计的浏览器扩展程序，旨在为同学们提供更加便捷的选课体验。通过集成历年课程评价信息，帮助学生更好地了解课程和教师情况，做出明智的选课决策。

## 主要功能

### 1. 按课程查询红黑榜：帮助同学查找最适合自己的课程
- **评论栏**：当前课程的教师评论信息
- **相关评论栏**：其他教师的相同或相似课程评论

### 2. 按教师查询：更加全面地了解教师的上课风格和教学特点
- **教师评论**：显示当前教师的所有课程评论
- **相关课程**：该教师的其他授课课程评论

### 3. 收藏功能：便于管理心仪的课程
- **一键收藏**：点击面板右上角⭐按钮即可收藏心仪课程
- **收藏夹管理**：统一管理所有收藏的课程信息
- **拖拽调整位置**：可自由拖拽收藏夹位置，方便查看
- **随时管理**： 收藏夹不受选课开放时间影响，随时随地都可以管理收藏夹

### 使用指南

1. **访问选课网站**
   - 登录南京大学选课系统

2. **查看课程信息**
   - 在课程列表中，点击课程名称或教师名称
   - 自动弹出信息面板显示相关评价

3. **切换查看模式**
   - **评论标签**：查看当前选择的课程/教师评论
   - **相关评论标签**：查看相关课程或该教师的其他课程评论

4. **收藏管理**
   - 点击面板右上角⭐按钮收藏课程
   - 收藏夹面板会自动显示在页面右上角
   - 可拖拽调整收藏夹位置

## 环境要求
- Chrome浏览器 88+ 或其他Chromium内核浏览器
- 支持Manifest V3的浏览器扩展环境

## 贡献与测试

欢迎为项目贡献代码！

可以使用我们提供的docker测试文件进行本地测试。使用方法如下：

在项目根目录下构建镜像并运行

```
docker build -f test/Dockerfile -t njuxkhelper-test .
docker run --rm -v "$PWD":/app -p 8000:8000 njuxk-test node serve_only.js
```

然后在本地浏览器打开`http://localhost:8000/test/index.html`即可。
